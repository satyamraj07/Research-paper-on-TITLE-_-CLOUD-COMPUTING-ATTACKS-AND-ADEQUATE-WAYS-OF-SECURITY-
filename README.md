## ☁️ What is Cloud Computing?

Cloud computing enables access to computing resources (servers, storage, databases) over the internet with scalability and flexibility.

### Service Models:

* **IaaS** – Infrastructure as a Service
* **PaaS** – Platform as a Service
* **SaaS** – Software as a Service
* **FaaS** – Serverless computing

---

## ⚠️ Types of Cloud Security Attacks

Based on the research, major threats include:

* 🚫 Denial-of-Service (DoS)
* 🔓 Account Hijacking
* 🧑‍💻 User Account Compromise
* 🦠 Malware Injection
* 👥 Insider Threats
* 📡 Side-Channel Attacks
* 🍪 Cookie Poisoning
* ⚙️ Security Misconfiguration
* 🔌 Insecure APIs
* ⛏️ Crypto Mining Attacks

These attacks can impact **confidentiality, integrity, and availability (CIA triad)**. 

---

## 🔐 Security Mechanisms

Key protection strategies include:

* 🔑 Identity & Access Management (IAM, RBAC)
* 🔒 Encryption (data at rest & in transit)
* 🛡️ Firewalls & IDS/IPS
* 📊 Security Monitoring (SIEM)
* 🔍 Logging & Auditing
* 📦 Backup & Disaster Recovery
* 📑 Compliance (ISO, NIST, GDPR)

## 🤖 AI/ML for Security (Anomaly Detection)

This project emphasizes **anomaly detection using machine learning**.

### Key Benefits:

* Detect unknown threats
* Reduce false positives
* Adapt to evolving attacks
* Automate incident response

### Workflow (Implementation Insight)

* Calculate dataset statistics
* Analyze patterns
* Detect anomalies using deviation
* Visualize abnormal behavior
